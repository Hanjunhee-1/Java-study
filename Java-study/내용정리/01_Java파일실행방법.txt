원래는 Eclipse 라는 IDE 를 통해 Java 프로젝트를 다루는 것이 훨씬 쉽다.

하지만, 뭔가 새로운 느낌을 주고 싶기도하고 이제는 vscode 가 더 편한느낌이어서...
vscode 로 Java 프로젝트를 다뤄보기로 했다.


일단 예전에 'vscode 로 Spring 코드를 작성할 수는 없을까?' 라는 생각에 vscode 에서도 Java 를 사용할 수 있게끔
세팅을 해두었었다. 혹시라도 다시 세팅을 해야되거나 처음부터 세팅을 해야되는 상황이라면 검색을 해보는 것을 추천한다.


생성한 Java 프로젝트의 디렉토리 구조를 보면 '/bin', '/lib', '/src' 이렇게 3개가 보인다.
'/src' 에는 '.java' 형식의 파일들이 모여있다. 
'/lib' 에는 의존성 파일들이 있는 곳이다. --> 솔직히 뭔 파일인지 아직은 잘 모르겠다.
'/bin' 에는 '.java' 형식의 파일들이 컴파일되어서 '.class' 형식의 파일들로 저장되는 곳이다.


★  java 로 작성한 파일 실행하는 방법!

    2가지 방법이 존재한다.
    1. '/bin' 위치로 이동하여 터미널에 'java [실행시킬파일명]' 을 입력한다.
    2. '/src' 에 있는 '.java' 형식의 파일을 직접 'Run' 버튼을 통해 실행시킨다.

    2가지 방법 중 어느 방법을 사용할지는 자유이다.
