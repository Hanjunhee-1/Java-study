변수를 선언할 때 타입을 지정해주어야 한다. 여러가지 타입이 존재하지만 자주 쓰이는 타입만 잘 알아두어도 좋다.

    -   숫자형
        (1) int 
            정수를 저장하기 위한 타입.
        (2) long
            int 와 마찬가지. 하지만 20억이 넘을 때는 long 타입을 쓴다.
        (3) float
            실수를 저장하기 위한 타입. 오차없이 7자리까지 저장 가능하다.
        (4) double
            float 와 마찬가지. 하지만 오차없이 15자리까지 저장 가능하다. 
    
    -   문자형
        (1) char
            문자를 저장하기 위한 타입.
        (2) String
            여러 문자(문자열) 를 저장하기 위한 타입.
